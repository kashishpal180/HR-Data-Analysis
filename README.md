# HR Data Analysis with Python

## About
This project analyzes HR employee data using Python. It includes data cleaning, exploratory data analysis (EDA), and visualizations to understand employee trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Basic HR Insights
